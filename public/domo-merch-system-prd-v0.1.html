01 · Overview
What We're Building

The Domo Cafe Merchandising System is an internal operations tool that gives the Domo Cafe team full visibility and control over its merchandise business — from the moment inventory arrives at the loading dock to the moment a guest takes a Domo plush home.

The system tracks inventory levels, sales performance, display placement, reorder triggers, and NHK royalty obligations — all in one place, purpose-built for a high-volume themed cafe retail environment.

The first order is already in: 4,788 units across 11 SKUs from License 2 Play. We need a system operational by opening day — June 13, 2026.

First Order Received: Order #231502-32715 · License 2 Play · 4,788 units · 11 SKUs · Shipped June 2, 2026. This order seeds the system's initial inventory state.
02 · Problem Statement
Why We Need This

Without a dedicated system, the merch operation faces several real risks at launch:

RiskImpactWithout System
Stockout on bestsellersLost revenue, guest disappointmentNo reorder alert until shelf is empty
Dead stock on slow moversCash tied up, floor space wastedNo sell-through visibility
NHK royalty errorsLicense compliance riskManual calculation, easy to miscalculate
Receiving errorsMissing units, vendor disputesNo PO-to-received reconciliation
Staff restock confusionFloor gaps, CDU emptiedNo planogram or restock schedule
No merch P&LCan't optimize the programRevenue buried in POS totals

04 · System Modules
Six Core Modules
📋
SKU Catalog
Master list of all merchandise. Every item: SKU, name, category, vendor, COGS, retail price, NHK royalty tier, case pack size, CDU or shelf unit.
Phase 1
📦
Inventory Tracker
On-hand count per SKU. Floor vs stockroom. Receiving log with PO reconciliation. Reorder thresholds. Low stock alerts.
Phase 1
🧾
Sales Log
Units sold per SKU per day. Revenue by category. Sell-through rate. Best sellers vs slow movers. Merch revenue per guest cover.
Phase 2
🗺️
Display Planner
Planogram: where each SKU lives — shelf, CDU, counter. Restock schedule per location. Visual floor map of the merch area.
Phase 3
🔄
Reorder Engine
When stock hits threshold → auto-generate PO draft. Vendor directory with lead times and minimums. Order history and packing slip archive.
Phase 3
🎌
NHK Royalty Calculator
Auto-split sales into tiers: 8% house merch / 0% licensed. Monthly royalty totals. Running MG tracker vs $35K floor. Reporting-ready export.
Phase 2
Analytics Dashboard (Phase 4): Best sellers ranked by units + margin · Blind box vs plush vs house merch revenue split · Weekly merch report · MG tracker · Merch attachment rate trends.
05 · User Stories
Who Uses This & How
P1As a manager, I want to see current stock levels for every SKU so I know what to restock before opening.
P1As a manager, I want an alert when any SKU drops below reorder threshold so I can place a PO before we run out.
P1As a receiver, I want to log incoming shipments against POs so we can confirm carton counts and flag discrepancies.
P1As a manager, I want to record daily merch sales by SKU so I can track what's selling and what isn't.
P2As a manager, I want to see monthly NHK royalty totals auto-calculated so I can stay compliant without manual math.
P2As a staff member, I want to know exactly which display unit each SKU belongs in so restocking is fast and consistent.
P2As Izzy, I want to see merch revenue by category (plush vs blind box vs house) so I can optimize the product mix.
P3As Izzy, I want weekly merch reports emailed or messaged to me automatically so I stay informed without logging in.
P3As Izzy, I want a running MG tracker so I know how close we are to the NHK minimum guarantee floor.
P3As a manager, I want to generate a PO draft from the reorder engine so I can send to L2P quickly.
