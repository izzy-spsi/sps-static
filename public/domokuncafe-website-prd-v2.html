Admin Login

Location

  • Route: `/admin` — not linked anywhere on the public site
  • Users access it by typing the URL directly
  • No breadcrumb, no footer link, no sitemap entry
  • Recommendation: consider a non-obvious slug like `/dc-admin` or `/manage` for light obscurity, but `/admin` is fine to start

Features

  • Email + password login
  • Session-based auth (JWT or cookie)
  • Dashboard: site content management (wait time, events, menu, CMS toggles)
  • Staff management (view all staff, roles, certifications)
  • Survey management (create, enable, disable surveys)
  • Activity log (who did what, when)

Access

  • Admin only — not shared with staff
  • Multi-admin supported (Izzy + store managers with admin role)

  • Name + role (e.g., "Welcome back, Hailey — Front of House")
  • Overall readiness score (composite of knowledge + service)
  • Certifications earned (badge display)
  • Active tasks / incomplete items
  • Recent activity

Navigation tabs:

TabContents
🏠 HomeDashboard, readiness score, quick actions
📚 LearnMenu modules, operating procedures, training materials
🎯 QuizzesKnowledge quizzes — must pass to advance
✅ CertificationsRoles earned, roles in progress, what's needed to unlock next
📝 My GrowthSurveys, self-assessments (manager-toggled), reflection prompts

---

Certification System

Two tracks required for each role:

1. Knowledge Track — digital

- Complete learning module for the role

- Pass quiz (score threshold, e.g., 80%+)

- Unlimited retakes, but tracked

- Auto-marks Knowledge Certified when passed

2. Service Track — manager-observed

- Manager observes staff performing the role in real life

- Manager marks Service Certified from their manager dashboard

- Cannot self-certify

Role examples:

  • Barista
  • Front of House
  • Food Prep
  • Cashier
  • Opening Shift
  • Closing Shift
  • Trainer (senior tier)

Certification badge states:

  • 🔒 Not started
  • 📖 Learning in progress
  • ✅ Knowledge certified
  • 🌟 Fully certified (knowledge + service)

---

Learning Modules

Each module contains:

  • Reading material (menu items, descriptions, allergens, prep notes)
  • Operating procedures (open/close checklists, service standards, hospitality rules)
  • Images / visual guides
  • End-of-module quiz

Modules are role-gated — staff only see modules relevant to their assigned roles.

---

Quiz System

  • Multiple choice + true/false
  • Pass threshold: 80%+
  • Retakes allowed — each attempt is logged
  • Manager can see attempt history
  • Passing auto-triggers Knowledge Certified status for that module

---

Surveys & Self-Assessments

  • Manager creates and enables surveys from Admin dashboard
  • Staff see active surveys under "My Growth" tab
  • Survey types:

- Open-ended reflection ("What went well this week?")

- Rating scales (1–5)

- Self-assessment rubrics ("Rate yourself on: Hospitality, Speed, Accuracy, Teamwork")

  • Responses visible to managers only
  • Staff can see their own history

---

Manager View

Team Overview:

  • Full staff list with readiness scores
  • Filter by role, certification status, quiz progress
  • Click any staff member → see their full profile

Actions managers can take:

  • Mark Service Certified for a staff member
  • Enable / disable surveys
  • Create quiz or survey assignments
  • Add roles to a staff member
  • View quiz attempt history
  • Add notes to a staff profile

Manager Dashboard widgets:

  • Team readiness %
  • Certifications this week
  • Pending service observations
  • Overdue incomplete items
